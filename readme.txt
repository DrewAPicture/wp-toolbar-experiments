== Description ==
This plugin is a WordPress core feature-plugin to test and iterate on proposed changes to the toolbar. Expanding on initial work done on core trac: https://core.trac.wordpress.org/ticket/32678.

Development is happening on GitHub: https://github.com/helenhousandi/wp-toolbar-experiments

If you're interested in contributing to this project, jump in on GitHub or join the #design channel in <a href="http://chat.wordpress.org/">WordPress core Slack</a>.

== Installation ==
1. Install through the WordPress plugin installer OR
1. Download the .zip file and upload the unzipped folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. The revisions to the toolbar will be visible in the admin and the front-end.
1. Leave feedback on the proposed changes on GitHub: https://github.com/helenhousandi/wp-toolbar-experiments.
